# Game Server Queue Simulation

A Discrete Event Simulation of a game server matchmaking queue.

## What it does
- Simulates players arriving at a game server
- Models queue formation when server is full
- Tracks wait times for each player
- Shows average and max wait time results

## How to run
python3 simulation.py

## Dataset
Online Gaming Behaviour Dataset from Kaggle
- 9000+ player records
- 13 columns including Age, Gender, GameGenre, PlayTimeHours

## Course
Simulation and Modelling — UMT Lahore 2026
## Sample Output
- 50 players simulated
- Server capacity: 8 players
- Players who waited: varies each run
- Average wait time: 3-5 seconds
- Simulation models peak hour gaming behaviour
